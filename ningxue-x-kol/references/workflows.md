# Workflows

## Daily Content Pack

When the user asks for today's content:

1. If fresh data is available, gather current X/trend/crypto/AI signals with `xint`, `last30days`, `surf`, web search, or user screenshots.
2. Select 3 themes:
   - One fast trend
   - One AI full-stack signal
   - One crypto/Web3 or cross-market signal
3. Produce:
   - 3-5 standalone posts
   - 5-10 replies to likely target posts
   - 1 thread outline
   - 1 recommended pinned/quote angle if relevant
4. Label each item with the pattern used.

## Rewrite Bio

Use the account identity:

- Keep "大冥灯" if possible.
- Include AI full-chain focus.
- Include A/H/crypto cross-market route.
- Include "边割边学".
- Include risk boundary and no leverage.
- Avoid sounding like a generic analyst.

Generate 5 variants:

- 原味版
- 专业版
- 锋利版
- 涨粉版
- 极简版

## Rewrite Pinned Post

Pinned post must answer:

- Who is this account?
- What does it track?
- Why should strangers follow?
- What is the risk boundary?

Keep a manifesto feel, but include a concrete follow reason.

## Generate Replies

For reply batches:

1. Identify the traffic node.
2. Choose reply pattern:
   - Chinese-context meme
   - One-line local joke
   - Serious event made everyday
   - Second-order scene observation
   - Visual meme reversal
3. Keep most replies under 30 Chinese characters.
4. Avoid overexplaining.
5. Provide safer and sharper variants when the topic is politically sensitive.

## Analyze User Posts

When the user provides screenshots of posts/replies:

1. Extract the text and visible metrics.
2. Classify the content pattern.
3. Explain why it worked or underperformed.
4. Add it to the pattern library if it is reusable.
5. Suggest a sharper version only if useful.

## Weekly Operating Rhythm

Daily:

- 5-15 high-quality replies to big traffic nodes.
- 2-4 short posts.
- 1 AI full-stack or crypto signal post.

Twice weekly:

- One deeper thread using detail-signal, power-order, or AI-radar format.

Weekly:

- One "大冥灯复盘" post:
  - What I got right
  - What I got wrong
  - What I am watching next

## Safety

Do not:

- Encourage leverage.
- Present market calls as certainty.
- Write paid pump-style content.
- Invent real-time market facts without checking.
- Use inflammatory political language as the main identity.

Prefer:

- "我现在的判断"
- "如果这个叙事要成立，需要满足..."
- "纯属个人复盘"
- "不构成任何投资建议"
